## tagColor
- x: gray
- b: blue
- r: red
- g: green
- o: orange
- y: yellow
  
## commands
- dailyTask:insert *insert a new task*
- dailyTask:clear *clear all task*
- dailyTask:reset *reset all task time zero*
- task item command: Edit Del SortUp SortDown SetTime

## add task
- tagColor#taskName#minute
- eg: `r#taskName#20`
